
# Employee Payroll Tax Calculator

A Python command-line program that calculates an employee's gross wages, federal tax, state tax, and net wages.

## Features

- Calculates gross wages from hours worked and hourly rate
- Applies federal tax based on marital status
- Applies state tax based on state of residence
- Calculates net wages after taxes
- Uses functions, parameters, and return values

## Python Concepts Used

- Functions
- Function parameters
- Return values
- Conditional statements
- User input
- Arithmetic calculations
- String conversion
